## General info
Note taking app

## Features
* Flexible Local Calendar
* Add events by importance level
* CRUP events (CREATE, READ, UPDATE, DELETE)
  
	
## Technologies
Project is created with:
* Flutter: 3.16.5
* Dart: 3.2.3
* Sql lite (https://pub.dev/packages/sqflite)
